# Demo Templates

This folder contains ready-to-customize README templates built with OctoMetrics widgets.

Templates included:

- `full-stack-engineer.md`
- `frontend-engineer.md`
- `backend-engineer.md`
- `devops-engineer.md`
- `data-ai-engineer.md`

Each template is a complete example that uses different combinations of:

- Banner widgets
- Typing widgets
- Stats cards
- Language cards
- Repository cards
- Streak cards
- Contribution widgets
- Dashboard widgets
- Metric badges
- Custom badge API examples

Replace the placeholder usernames, links, titles, and stack details with your own values before using them in a profile README.
